# Lua Arcade (LÖVE)

Один проект на Lua + LÖVE (LOVE2D), в котором будет 3 игры:
- Snake
- Minesweeper
- Pac-Man (позже)

## Требования
- Установленный LÖVE (LOVE2D)

## Запуск
Открой папку проекта через LÖVE или перетащи папку на love.exe.

## Управление
- Меню: мышь / стрелки / Enter
- Игра: WASD или стрелки
- Esc: пауза/назад
- R: рестарт

## Структура
- `main.lua`, `conf.lua` — вход и настройки LÖVE
- `src/` — код сцен и игр
